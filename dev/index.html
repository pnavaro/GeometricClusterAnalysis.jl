GeometricClusterAnalysis.jl

Documentation for GeometricClusterAnalysis.jl

This set of functions is not a Julia package. Here you'll find a series of examples and methods that can be used to approximate compact subsets in noisy datasets. It could be use for clustering too.

Installation

To use this package, you first need to install Julia.

Clone this repository to get the library and all examples:

git clone https://github.com/pnavaro/GeometricClusterAnalysis.jl
cd GeometricClusterAnalysis.jl

Install dependencies

$ julia --project
